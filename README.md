# Manual-Testing-Project-End-to-End-QA-Artifacts
This repository contains complete manual testing artifacts for a real-time web application, covering functional, regression, and end-to-end testing.

🔹 Manual Testing Key Artifacts 

1. Test Plan
- Scope, objectives, timelines
- Testing types (Functional, Regression, UAT)
- Risks & mitigation

2. Test Scenarios & Test Cases
- Positive / Negative scenarios
- Boundary conditions
- Reusable steps

3. RTM (Requirement Traceability Matrix)
- Requirement → Test Case mapping
- Ensures 100% coverage

4. Defect Log
- Bug ID, Steps, Severity, Priority
- Status tracking (Open → Closed)

5. Execution Report
- Pass/Fail status
- Screenshots/evidence
  
6. Test Summary Report
- Total test cases
- Pass %, Fail %, Defects
- Release recommendation
